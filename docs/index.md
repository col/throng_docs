# Throng Docs

Welcome. This is the home page for the Throng platform documentation.

## Getting started

Edit `docs/index.md` (this file) to change what you see here. Add new pages
by creating more `.md` files inside `docs/` — they'll show up in the sidebar
automatically.

!!! tip "Live preview"
    Run `uv run mkdocs serve` from the project root and open
    <http://127.0.0.1:8000> — the page reloads as you edit.
